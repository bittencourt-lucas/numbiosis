Método de Gauss Jordan

Método numérico usado para resolver sistemas de equações lineares

Texto

O método de Gauss-Jordan é um método de escalonamento que consiste em aplicar operações elementares à matriz aumentada de um sistema, até que ela esteja na forma escalonada reduzida. A vantagem deste processo é que um sistema cuja matriz aumentada é uma matriz na forma escalonada reduzida tem solução imediata, enquanto que para resolver um sistema que está apenas na forma escalonada ainda é necessário fazer uma série de substituições para obter a soluçãoo final.

Definição.

Uma matriz está na forma escalonada reduzida quando ela satisfaz as seguintes condições:

  1. O primeiro elemento não-nulo de cada linha não-nula (chamado o pivô da linha) é igual a 1.
  2. O pivô da linha i + 1 ocorre à direita do pivô da linha i.
  3. Se uma coluna contém um pivô, então todas os outros elementos desta coluna são iguais a 0.
  4. Todas as linhas nulas ocorrem abaixo das linhas não-nulas
